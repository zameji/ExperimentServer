Task Batteries

Shifting (UZH-Shifting)

This task battery comprises four tasks for each of five shifting components judgment, dimension, mapping, response set, and stimulus set switching. Each set of tasks comprises two two-choice, one three-choice, and one four-choice task.

Background

Shifting attention between mental task sets has been extensively researched, but it is still remains unclear what constitutes the assumed underlying shifting ability. Therefore, the UZH-Shifting battery was used to study the commonalities and differences of five task switching components (i.e., switching between varying judgments, stimulus dimensions, stimulus-response mapping, response sets, and stimulus sets). The tasks were modelled after paradigms commonly utilized in the experimental and individual differences literature on task switching. They were originally programmed in Java-based Tatool.
