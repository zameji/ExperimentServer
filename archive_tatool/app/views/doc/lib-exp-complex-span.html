Experimental Tasks

Complex Span

In the Complex Span paradigm, participants have to switch between memorising stimuli, and responding to a choice-reaction time task. After a series of such switches, they're asked to recall the stimuli memorised. In Tatool, this is implemented as Dual Element composed of the Memory Span and the Choice Reaction Time paradigms.

Element

Use a Dual Element as the parent element in order to create the Complex Span paradigm.

Executables

Project tatool

Project tatool
Executable Type tatoolChoiceReaction

The Memory Span Executable has to be used as the primary Executable of a Dual Element in order for it to work properly.
