Template Services

Tatool offers a set of template services that are useful when you develop your own Executables or Handlers. The services offer methods that can be used to arrange, display and hide stimuli on-screen or to allow user input by keyboard and/or mouse.

To use any of the template services, you need to add the respective dependency to your Executable Service. The following code shows you an example that uses the template service called stimulusServiceFactory in your Executable Service:

myExecutable.service.js

tatool.factory('myExecutable', [ 'executableUtils', 'stimulusServiceFactory',
  function (executableUtils, stimulusServiceFactory) {  

    var MyExecutable = executableUtils.createExecutable();

    MyExecutable.prototype.init = function() {
      this.stimulusService = stimulusServiceFactory.createService(this.stimuliPath);
    };

    // our custom methods go here

    return MyExecutable;
  }]);
