Documentation

Welcome to the Tatool documentation, your resource to learn everything you need to know to start using Tatool for your own experiments. If you find errors in the documentation or if you think something important is missing, please don't keep it to yourself but tell us!

The documentation is split into the following parts:

  • Start: this page and the glossary
  • About: learn about the idea and people behind Tatool
  • Community: connect with other people using Tatool and find help
  • Using Tatool Web: introduction to the basic functionality of Tatool
  • Task Library: reference of available tasks and how to use them
  • Development: learn how to create your own tasks
  • Reference: learn about specific methods and attributes while developing
