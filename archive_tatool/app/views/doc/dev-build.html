Building Tatool Web

Tatool Web uses webpack for its build process automation. Details about the loaders and plugins can be found in the webpack config files.

Run Tatool with webpack-dev-server, if you want to extend the functionality of tatool while using automatic reloading. You'll have to run the server in a separate process.

# run server
> npm start

# run webpack-dev-server
> npm run dev

Build tatool with webpack.


# install dependencies and build tatool
> npm run setup

Run Tatool in lab mode, which deactivates the registration and email features and adds a default admin user.

$> npm start

# alternatively you can call node directly
> node server.js lab prod

If you want to run Tatool Web in server mode, you'll have to provide the following two configuration parameters. The environment parameter will direct Tatool Web to enable or disable server logging.

# [mode] : lab / server
# [environment] : dev / prod
> node server.js [mode] [environment]
