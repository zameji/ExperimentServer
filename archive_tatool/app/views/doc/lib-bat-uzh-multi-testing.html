Multifacet Working Memory Testing and Training Package (UZH-Multi)

Testing

This set of tasks can be used to measure working memory capacity before and after training. Relational integration (cf. Oberauer et al., 2008) is measured by the Tower of Fame task. In this task, sequentially presented sentences contain information about which famous person resides in which of four apartments across four floors in relation to the aforementioned person. After a variable number of such sentences, the task is to recall who lived where.

Storage and Processing, and Supervision are measured by a complex span and an switching task, respectively. Both tasks will soon be added to the module.

Tower of Fame Task

Executable

Use the following settings in the Module Editor for these tasks:

Project uzh-multi-battery
Executable Type towerOfFame

Properties

The following Properties can be provided in the Module Editor to change the behaviour of the task (note that, except for displayLanguage, the default value is the one used in the publication):

Property Name Property Type Default Mandatory Description
stimuliFile Resource None, see Stimuli Files below for options Yes A stimuli file expecting stimuli according to the Tatool Stimulus definition.
stimuliPath Path None, use the following:
Project: uzh-multi-battery
Type: stimuli		 Yes A stimuli path where to load any resources from (stimuli images, input key images).
displayDuration String 4750 No Timer duration in ms to wait for user feedback before stopping.
intervalDuration String 250 No Timer duration in ms to wait and present a blank screen before the next sentence.
displayLanguage String EN No Language in which stimuli are presented (EN for English, DE for German)
