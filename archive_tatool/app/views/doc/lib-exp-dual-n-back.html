Experimental Tasks

Dual N Back

In the Dual N-Back paradigm, participants need to respond whenever the current stimulus was the same as the one presented N positions back in the sequence. Dependent measure is the proportion of hits minus false alarms averaged over both modalities and all n-back levels

Element

Use a List Element as the parent element in order to create the Dual N-Back paradigm. The number of iterations should be set to the number of stimuli to be shown.

Executables

Use the following settings in the Module Editor for this type of task:

DualNBack

Project hsrc
Executable Type DualNBack

Properties

The following Properties can be provided in the Module Editor to change the behaviour of the task:

displayDuration The time in milliseconds to display the stimulus
nValue The value of N refered to in the initial description
iterations This should match the iterations in the list element. Used to assist in generating the correct number of targets
targets How many target stimuli should be generated (default is 12)

Stimuli File

dual-nback-permutations.csv contains a list of the available stimuli. This lists all the visual and audio stimuli also in the same folder.
