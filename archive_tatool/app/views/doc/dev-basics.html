What Skills Do I Need?

You don't need any superpowers in order to run a local version of Tatool in your own lab. Simply follow the instructions on our Getting started page and you should be up and running in no time.

If you want to start developing your own tasks, you'll need to make yourself familiar with JavaScript, HTML, and CSS. These are the technologies you'll be working with when programming with Tatool. There are tons of free tutorials on the web which can get you started quickly. For example, Codecademy offers gentle and interactive introductions:

  • JavaScript
  • HTML and CSS

That's all you need!
