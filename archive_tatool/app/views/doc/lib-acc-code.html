Accessory Executables

Code

The Code Executable provides the functionality for a user to enter his/her unique experiment participant code. This comes in handy, when you want to run an experiment in your lab without creating a separate user for every participant. The code entered by the participant will be available as the session property sessionCode in your data export.

Executable

Use the following settings in the Module Editor for this type of task:

Project tatool
Executable Type tatoolCode

Properties

The following Properties can be provided in the Module Editor to change the behaviour of the task:

Property Name Property Type Default Mandatory Description
inputText String 'Please enter your Code below:' No The text that will be displayed on screen to instruct the user to enter their code.
exactCodeLength String - No The exact number of required characters/digits for the code. If the code entered doesn't fulfill this rule, a warning icon is displayed and the user can try again.
minCodeLength String - No The minimum number of required characters/digits for the code. If the code entered doesn't fulfill this rule, a warning icon is displayed and the user can try again.

Stimuli File

This task does not support the use of a stimuli file.
