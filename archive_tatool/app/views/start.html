Flexible Standards

The new Tatool is based on the widely-used and well-known web technology standards HTML5, CSS, JavaScript and node.js. This makes it easy to create your own experiments even with only little prior programming knowledge.

Built for researchers

We focused on the functionality that you need to get your research done. The new Tatool assists you throughout the process: Go from programming and configuring your experiments to collecting data and analysing it, in no time.

Reach everyone, everywhere

Thanks to Tatool's web technologies you can test participants from all around the world by telling them to just start up their favourite web browser. You prefer collecting data offline? No problem, Tatool runs also in offline lab-mode.
