Modules

This section introduces the functionality of the Tatool Web platform in order to get you started quickly. Every experiment is unique but usually follow a similar pattern on how it is conducted. Tatool Web provides the functionality to make your data collection as easy as possible. We suggest you read through the entire Using Tatool Web part of the documentation to get an idea of what Tatool Web can do for you.

Feel free to post any questions you might have about how you intend to use Tatool Web for your study on our Github page.

Depending on your assigned user roles, you can see and use the following parts (available as tabs in the top navigation after login) of Tatool:

  • Modules: Add and run Modules. The Modules are either publically available or you've been invited to participate in a private Module.
  • Editor: Create and edit your own Modules with the help of the convenient Module Editor. After you've created your Module, you can easily publish Modules to the Tatool repository for users to run your Modules (only available to researchers).
  • Analytics: Use the Analytics to monitor participation in your experiment with detailed data for every session (only available to researchers).
  • Admin: The Admin can manage users and projects through a convenient user interface (only available in your local installation).
