Language Use and Cognition (UZH-LuCo)

Linguistic Ability

This set of tasks measures aspects of linguistic ability. In the Lexical Decision Making Task, strings of letters are presented. For each letter string, the task is to press the left arrow key when the string is a word and the right arrow key when it is a non-word.

In the Word Recognition Task, sequentially presented nouns have to be memorised. In the subsequent recognition phase, those previously presented words are intermixed with new words. The task is to indicate which words were presented previously and which were not.

Executable Type tatoolChoiceReaction

See Choice Reaction Time Task in the Task Library for a description of the Properties that can be provided in the Module Editor to change the behaviour of the task.
