Generic Executables

You can combine and customise the Generic Executables to create your own experiments and task batteries. You can select from Experimental Tasks (customisable paradigms, typically from the cognitive, neuroscientific, and/or social psychological literature), Adaptive Training Tasks (paradigms allowing for adjusting task difficulty to individual performance), and Accessory Executables (anything else you need to run a study, such as Executables to present instructions).
