Utility Services

Tatool offers a set of utility services that are useful when you develop your own Executables or Handlers. The utility services offer methods that can be used to access files, prepare stimuli, store trials, and more. You can also create your own services, offering custom reusable code for your Executables or Handlers. Currently, the utility services comprise dbUtils, executableUtils, statusPanelUtils, and timerUtils.

To use any of the utility services, you need to add the respective dependency to your Executable Service. The following code shows you an example that uses the utility service called dbUtils in your Executable Service:

myExecutable.service.js

tatool.factory('myExecutable', [ 'executableUtils', 'dbUtils',
  function (executableUtils, dbUtils) {  

    var MyExecutable = executableUtils.createExecutable();

    MyExecutable.prototype.init = function() {
      // we can start using the dbUtils service here
      dbUtils.setSessionProperty(this, 'currentValue', 12);
    };

    // our custom methods go here

    return MyExecutable;
  }]);
