statusPanelUtils

The statusPanelUtils service offers methods that allow you to interact with the Tatool status panels.

  • Status Panel Methods

Status Panel Methods

When to Use the Status Panel Methods
Usually, you'll not have to manually update a status panel value, as Tatool takes care of updating the status panel by its own or by the help of customised handlers. See the methods below to learn how and when to use them.

setFeedback(score) - (no return value)

Used to manually update the feedback status panel to display a visual feedback on whether the response given by the user was correct or wrong. Note that the feedback status panel is automatically updated when a Trial object is saved with the dbUtils service. Use the setFeedback() method when you don't save a Trial or want to update the feedback at another custom time during your Executable.

Parameters
score: Integer value indicating the score (> 0 will show a correct feedback and 0 will show a wrong feedback).

Example

// update the feedback within an Executable
statusPanelUtils.setFeedback(1);
