levelHandler

The levelHandler provides the values for the level status panel and can be set for every Executable.

The levelHandler always retrieves the latest level, counter, score at the beginning of a Session. During a Session, the counter executes the level algorithm after every execution of an Executable, regardless of whether a Trial object is saved. In the case of a Dual Element, the algorithm will only be executed after one complete iteration (i.e., primary, secondary, primary). The level status panel is updated at the beginning of every new execution and should therefore show the current level.

Handler Properties

The following Properties can be provided in the Module Editor to change the behaviour of the handler.

Property Name Property Type Default Mandatory Description
benchmarkSampleSize String 3 No Number indicating the interval of when the level algorithm should compute the performance and trigger a level-up/level-down event (number of trials).
benchmark String 0.6 No Float value between 0 and 1 indicating what percentage of correct answers (performance = totalScore / maxScore) must be achieved for a level-up. If performance greater or equal to the benchmark, a level-up will be triggered. If performance is lower than the benchmark a level down will be triggered (depends on Property allowLevelDown).
allowLevelDown Boolean false No Boolean value indicating whether level down is triggered (true) or not (false).

Module/Session Properties

The levelHandler stores the following Module Properties to keep track of the level information across multiple sessions:

Property Description
counter Counter of how many trials have been executed. Used to keep track of when to trigger the level algorithm according to benchmarkSampleSize.
totalScore The summed up total score achieved over the last interval of trials (benchmarkSampleSize). Source of the value is the score property of the stored trial object.
maxScore The summed up maximum score possible over the last interval of trials (benchmarkSampleSize). The maxScore is set to 1 by default for every trial.
currentLevel The current level of this levelHandler
