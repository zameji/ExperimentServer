Accessory Executables

Countdown

The Countdown Executable displays a numerical countdown that you can use to make sure the user is ready at the beginning of your Executables.

  • Executable
  • Properties
  • Stimuli File
  • Download

Executable

Use the following settings in the Module Editor for this type of task:

Project tatool
Executable Type tatoolCountdown

Properties

The following Properties can be provided in the Module Editor to change the behaviour of the task:

Property Name Property Type Default Mandatory Description
countdown String 5 No The number to start to count down from.
interval String 600 No Interval used to refresh countdown numbers.
goText String 'Go!' No Text that's displayed once the countdown has finished.

Stimuli File

This task does not support the use of a Stimuli File.
