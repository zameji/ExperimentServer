Shifting (UZH-Shifting)

Mapping Switching

This set of tasks measures shifting between different stimulus-response mappings. The table below lists the four mapping switching tasks available.


Variant Task pair Description Adapted from
Mapping 1 Autumn/Winter Classify the color of simple shapes. Meiran (1996)
Mapping 2 Circle/Triangle Decide whether two simultaneously presented fractal images are the same or different. von Bastian & Oberauer (2013)
Mapping 3 Red/Green Classify a word as city, river, or country. von Bastian & Oberauer (2013)
Mapping 4 Dark/Light Identify the mathematical symbol presented. Duncan (1978)

Module

The following module comprises all four tasks: shifting_mapping.json

Note: This module presents the tasks in the order as they are listed in the above table, not in the order in which they were run in the published study.

Executable

Use the following settings in the Module Editor for these tasks:

Project uzh-shifting-battery
Executable Type uzhShifting

Properties

The following Properties can be provided in the Module Editor to change the behaviour of the task (note that the default value is the one used in the publication):

Property Name Property Type Default Mandatory Description
stimuliFile Resource None, see Stimuli Files below for options Yes A stimuli file expecting stimuli according to the Tatool Stimulus definition.
stimuliPath Path None, use the following:
Project: uzh-shifting-battery
Type: stimuli		 Yes A stimuli path where to load any resources from (stimuli images, input key images).
displayCueDuration String 150 No Timer duration in ms to display the cue before target stimulus onset.
displayDuration String 2000 (not set in publication) No Timer duration in ms to display the target stimulus.
